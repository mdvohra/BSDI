## Model weights

Large checkpoints (`.pth`, `.pt`, `.safetensors`, GeoTIFF samples, etc.) are **not** in this repo. Place them under `models/artifacts/` and related paths as described in `models/**/WHAT_GOES_HERE.txt` files.
